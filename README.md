# DocSnap: API Documentation Generator

**DocSnap** is a tool to create API documentation from your Postman collections. You can import collections using a UID or upload your own JSON. The documentation can then be exported as a PDF, making it easy to share.

**LINK:** https://docsnap-ttyn.vercel.app/

## Features

- **Import Postman Collection**: You can import Postman collections by using a collection UID.
- **Search and Filter**: Quickly search for requests by name, URL, or HTTP method.
- **Export to PDF**: Download the API documentation as a PDF.
- **Path Variables and Query Params**: Automatically detects and shows path variables and query parameters in requests.
- **Response Body**: Shows the response body in a clean and readable format.
- **Supports Multiple HTTP Methods**: Includes support for GET, POST, PUT, DELETE, PATCH, etc.
- **Easy to Use**: Simple user interface for managing API docs.

## How It Works

1. **Import a Postman Collection**:
   - Use the collection UID from Postman to import your API documentation.
   - You can also upload a Postman collection file (JSON format).
  
2. **View and Explore Requests**:
   - Once the collection is imported, view the details for each request, including:
     - Request Method (GET, POST, etc.)
     - Request URL
     - Headers, Query Params, and Request Body
     - Response details
  
3. **Search and Filter**:
   - You can search for any request by name, URL, or HTTP method.
  
4. **Export to PDF**:
   - After reviewing your API documentation, you can export it as a PDF file.
   - The PDF will include all the request details, headers, bodies, and responses.
